Geheugen

RAM (Random Access Memory)

RAM staat voor Random Access Memory. RAM wordt gebruikt door je computer om tijdelijke data op te slaan die momenteel in gebruik is. Ram zorgt ervoor dat je computer sneller en beter werkt.
Hoe meer RAM je computer heeft, hoe beter je computer kan omgaan met meerdere programma's tegelijkertijd. RAM is veel sneller dan je harde schijf. Het geheugen wordt door de processor gebruikt om snel toegang ye krijgen tot de gegevens die nodig zijn voor de programma's. Dit maakt je computer sneller. RAM is alleen actief wanneer de computer aanstaat.

ROM (Read Only Memory)

ROM staat voor Read-Only Memory. In de ROM worden gegevens opgeslagen die niet snel gewijzigd kunnen worden. De inhoud van ROM kan alleen gelezen worden. In tegenstelling tot RAM, blijft de informatie van ROM wel bewaard als de computer afgesloten wordt. ROM wordt vaak gebruikt om systeemsoftware op te slaan, zoals de BIOS (Basic Input/Output System).
